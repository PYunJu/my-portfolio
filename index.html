PlanIt

Made with:

A collaborative mobile app designed for group travel planning. Users can create trip plans, add schedules, and invite friends to participate. Features include a shared calendar view, real-time group chat, and voting functionality for making group decisions on destinations or activities. The UI emphasizes clarity and ease of use, optimized for mobile devices. Real-time features are implemented via socket communication. EcoMeter

Made with:

An eco-conscious web application that helps users monitor and reduce their carbon footprint. The platform tracks daily activities such as transportation, energy usage, and recycling, then visualizes their environmental impact over time through dynamic charts. Users can set goals, compare their carbon output across weeks or months, and receive actionable tips on how to live more sustainably. The backend uses Django and REST API to deliver structured environmental data, while Chart.js handles the rendering of visual data. The app runs in Docker containers and stores user data securely on AWS RDS.
